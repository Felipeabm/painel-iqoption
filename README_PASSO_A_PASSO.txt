
# PAINEL DE SINAIS PARA IQ OPTION - GUIA

1. Crie um repositório no GitHub e envie todos os arquivos.
2. Crie uma conta gratuita no Render.com
3. Escolha "Web Service", conecte ao repositório e selecione:
   - Build Command: pip install -r requirements.txt
   - Start Command: python app.py
4. Acesse o link gerado pelo Render em qualquer dispositivo.

Login exigirá e-mail e senha da sua conta demo da IQ Option.
